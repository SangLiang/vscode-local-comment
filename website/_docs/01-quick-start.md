# 快速开始

欢迎使用 Local Comment！本指南将帮助你在 3 分钟内完成首次本地注释的添加。

<div class="callout callout-tip">
<strong>提示：</strong>Local Comment 的所有数据默认存储在项目目录下的 <code>.vscode/local-comment/</code> 中，不会修改你的源代码文件。
</div>

## 安装扩展

1. 打开 VS Code:
2. 点击左侧活动栏的「扩展」图标（或按 <kbd>Ctrl+Shift+X</kbd>）
3. 搜索 "Local Comment"
4. 点击「安装」

## 添加第一条注释

1. 在编辑器中打开任意代码文件
2. 将光标放在你想添加注释的代码行上
3. 按 <kbd>Ctrl+Shift+M</kbd>（Mac: <kbd>Cmd+Shift+M</kbd>）
4. 在弹出的 Markdown 编辑器中输入你的注释内容
5. 按 <kbd>Ctrl+Enter</kbd> 保存并关闭

<div class="callout callout-tip">
<strong>新手提示：</strong>建议把注释添加在「有意义的代码行」上，例如函数声明行，这样即使代码发生变更，注释也更容易跟随定位。
</div>

## 查看注释列表

1. 打开左侧活动栏的「Local Comments」视图
2. 你会看到当前项目中的所有注释列表
3. 点击任意注释项，编辑器会自动跳转到对应的代码位置

## 添加书签

1. 将光标放在目标代码行
2. 按 <kbd>Ctrl+Alt+K</kbd> 添加书签
3. 按 <kbd>Ctrl+Alt+J</kbd> 跳转到下一个书签
