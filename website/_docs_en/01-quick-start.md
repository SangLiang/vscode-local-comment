# Quick Start

Welcome to Local Comment! This guide will help you add your first local comment in 3 minutes.

<div class="callout callout-tip">
<strong>Tip:</strong> All Local Comment data is stored in <code>.vscode/local-comment/</code> under your project directory by default, and will not modify your source code files.
</div>

## Install the Extension

1. Open VS Code:
2. Click the "Extensions" icon in the left Activity Bar (or press <kbd>Ctrl+Shift+X</kbd>)
3. Search for "Local Comment"
4. Click "Install"

## Add Your First Comment

1. Open any code file in the editor
2. Place the cursor on the line where you want to add a comment
3. Press <kbd>Ctrl+Shift+M</kbd> (Mac: <kbd>Cmd+Shift+M</kbd>)
4. Enter your comment content in the Markdown editor
5. Press <kbd>Ctrl+Enter</kbd> to save and close

<div class="callout callout-tip">
<strong>New user tip:</strong> It's recommended to attach comments to "meaningful code lines", such as function declaration lines. This way, even if the code changes, the comment is more likely to follow and stay anchored.
</div>

## View Comment List

1. Open the "Local Comments" view in the left Activity Bar
2. You will see a list of all comments in the current project
3. Click any comment item, and the editor will automatically jump to the corresponding code location

## Add a Bookmark

1. Place the cursor on the target code line
2. Press <kbd>Ctrl+Alt+K</kbd> to add a bookmark
3. Press <kbd>Ctrl+Alt+J</kbd> to jump to the next bookmark
