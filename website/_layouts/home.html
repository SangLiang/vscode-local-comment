为什么选择 Local Comment？

📝

本地注释

不污染源码的持久笔记，独立于 Git 分支。

🔖

书签导航

跨文件的阅读路径，快速定位关键代码。

🏷️

标签跳转

${声明} 与 @引用 双向链接，支持中文。

🎨

Markdown 渲染

支持 Mermaid 图表、LaTeX 公式和语法高亮。

💾

数据隔离

项目级存储，支持多组注释配置自由切换。

⌨️

快捷键驱动

全键盘操作，不离开编辑器即可管理注释。

30 秒快速开始

1

安装扩展

在 VS Code: 扩展市场搜索 "Local Comment" 并安装。

2

添加注释

将光标放在代码行上，按 Ctrl+Shift+M 打开 Markdown 注释编辑器。

3

查看列表

打开侧边栏「Local Comments」视图，查看和管理所有注释。
